## 发布订阅模式

```javascript
class EventBus {
  constructor() {
    this.eventMap = new Map();
  }

  on(eventName, cb) {
    // eventMap中不存在此事件，则新建此事件
    if (!this.eventMap.has(eventName)) {
      this.eventMap.set(eventName, [{ cb, isOnce: false }]);
      return;
    }

    this.eventMap.get(eventName).push({ cb, isOnce: false });
  }

  emit(eventName, ...args) {
    // 没有此事件，则直接return
    if (!this.eventMap.has(eventName)) return;

    const arr = this.eventMap.get(eventName);

    arr.forEach(({ cb }) => {
      cb.apply(undefined, args);
    });

    this.eventMap.set(
      eventName,
      arr.filter(({ isOnce }) => !isOnce)
    );
  }

  once(eventName, cb) {
    if (!this.eventMap.has(eventName)) {
      this.eventMap.set(eventName, [{ cb, isOnce: true }]);
      return;
    }

    this.eventMap.get(eventName).push({ cb, isOnce: true });
  }

  off(eventName, fn) {
    if (!this.eventMap.has(eventName)) return;

    const arr = this.eventMap.get(eventName);

    const index = arr.findIndex(({ cb }) => fn === cb);

    if (index >= 0) {
      arr.splice(index, 1);
    }
  }
}
```

## 观察者模式

两个类：主题（被观察者）、观察者
主题（被观察者）：

1. 添加观察者
2. 移除观察者
3. 通知观察者们

观察者：

1. 被通知时作出的反应

```javascript
class Subject {
  constructor() {
    this.observers = [];
  }

  subscribe(observer) {
    if (!this.observers.find((item) => item === observer)) {
      this.observers.push(observer);
    }
  }

  unsubscribe(observer) {
    this.observers = this.observers.filter((item) => item !== observer);
  }

  notify(data) {
    this.observers.forEach((item) => {
      item.update(data);
    });
  }
}

class Observer {
  constructor(name) {
    this.name = name;
  }

  update(data) {
    console.log(`${this.name}收到数据：${data}`);
  }
}
```
